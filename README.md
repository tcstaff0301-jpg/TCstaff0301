# Orange Cat Pic Skill

一个用于中文文章正文配图的 Codex skill。

它会把文章里的关键观点、流程、结构、状态或隐喻，转成 16:9 横版的橘猫正文配图。默认风格是：Igloo 奶油白背景、Tangerine 橘猫 IP、Brunnera / Cadet Blue / Black Magic 多色组合、手绘批注、大量留白、怪诞但清爽的产品草图感。

## Trigger

推荐触发词：

```text
橘猫配图
```

示例：

```text
橘猫配图，为下面这篇文章设计 5 张正文配图，先给 shot list。
```

```text
橘猫配图，为这段内容生成一张 16:9 正文配图。
```

## Notes

这不是 PPT 信息图 skill，也不是儿童贴纸生成器。它的目标是为中文长文、公众号、Notion、飞书文档、小红书图文等内容生成“看起来有点怪，但 1 秒能懂”的正文解释图。
